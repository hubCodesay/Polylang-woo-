# Polylang WooCommerce Bridge

WordPress plugin that connects Polylang with WooCommerce:

- product translations
- product taxonomies translations (categories, tags, shipping classes, `pa_*` attributes)
- language-aware WooCommerce page mapping (shop/cart/checkout/account)
- optional automatic header language switcher

## Requirements

- WordPress 6.0+
- PHP 7.4+
- WooCommerce active
- Polylang active

## Install

1. Copy `polylang-woocommerce-bridge` to `wp-content/plugins/`.
2. Activate plugin.
3. Save permalinks once (`Settings -> Permalinks`).
4. Configure switcher options in `Settings -> General`.
